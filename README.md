# Amazon India — Product Analysis Dashboard

**Overview:**
This project involves a comprehensive analysis of an Amazon India product catalog dataset to extract actionable business insights. The primary objective was to visualize key performance metrics such as pricing strategies, discount impact, customer ratings, and product distribution, enabling better sales and inventory management decisions.

**Key Features:**
*   **Data Processing:** Performed data cleaning and organization of the dataset to ensure analytical accuracy.
*   **Business Intelligence:** Utilized advanced Excel features and Python/Pandas to derive key metrics and category-specific trends.
*   **Data Visualization:** Created an interactive dashboard providing insights into pricing, ratings, and market positioning.

**Tools & Techniques:**
*   **Software:** Microsoft Excel, Python, Pandas.
*   **Methodology:** Data Analysis, Dashboard Design.
